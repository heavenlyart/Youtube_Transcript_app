# Youtube_Transcript_app
This is a app created in python that can be used to Generate Transcript of a youtube video if available in english with the video Id. 
## Installation ⚙️

### Prerequisites

◉ Make sure you have Python 3.x installed on your system.

### Requirements

◉ To install the necessary dependencies, use the `requirements.txt` file included in the project. You can install them using pip:

```bash
pip install -r requirements.txt
```
### Running the Application

1. Clone this repository to your local machine.
2. Navigate to this folder in your terminal.
3. Run the application:

```bash
python Main.py
```
